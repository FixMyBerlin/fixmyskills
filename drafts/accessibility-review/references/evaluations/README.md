## Evaluation criteria

Each file covers: what it is, structure, pros, cons, relevance to our German public-sector draft, verdict (adopt / imitate / skip), and content worth copying.

**Verdict key**

- **adopt** — install or merge substantially as-is
- **imitate** — borrow patterns, checklists, or structure into our skill
- **skip** — low marginal value or poor fit

**Status:** eleven repos evaluated 2026-07-09 (four baseline + seven agent-workflow/runtime).
